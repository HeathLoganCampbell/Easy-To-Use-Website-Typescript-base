# `Pixijs + Typescript`
Quickly and easily have all the perks of typescript without any of the complexity.

* Compiles and bundles ts files with source mappings
* hot reloads
* browser syncs
* Typescripts importing enabled

## How to run
```
npm i
npm start
```
